  Loom: Self-service provisioning for microservices in Kubernetes/AWS

Do you want your development teams to be able to quickly create microservices, without depending on operations? Loom lets your developers provision the cloud resources they need, automatically, without manual work by ops.

Developers use Loom's RESTful API to request specific resource types. Ops engineers configure the Loom back-end to map resource requests to organization-specific configurations.

Loom is sponsored by Project Blackbird, helping you deploy microservices on Kubernetes in AWS
